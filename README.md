Este proyecto es un portfolio web personal desarrollado para mostrar de forma clara y moderna quién soy, qué tecnologías manejo y en qué tipo de proyectos estoy trabajando.[1][2]

## Tecnologías utilizadas

- HTML5 y CSS3 para la estructura semántica y el estilo base de toda la página.[2]
- Bootstrap 5 para conseguir un diseño responsive rápido, con un grid limpio y componentes coherentes en todas las secciones.[2]
- JavaScript para la lógica de la página (interacción, validaciones, carga dinámica de componentes y comportamiento del formulario de contacto).[3]
- EmailJS para enviar el formulario de contacto directamente a mi correo sin necesidad de backend propio, manteniendo el proyecto como estático pero funcional a nivel profesional.[4][5]
- n8n para automatizar procesos relacionados con el chatbot y algunas integraciones (por ejemplo, clasificación de mensajes y conexión con otros servicios).[6][7]

## Diseño y estructura

- One-page layout con secciones claras: inicio, sobre mí, habilidades, experiencia, proyectos y contacto, pensado para que un reclutador o cliente vea toda la información clave en pocos scrolls.[8][2]
- Estética oscura con acentos de color y botones degradados, para transmitir un estilo moderno y tecnológico que encaje con el perfil de desarrollador full stack.[9]
- Uso intensivo de iconos y logotipos de tecnologías (frontend, backend y herramientas) para que se identifiquen rápidamente las stacks dominadas sin necesidad de leer demasiado texto.[9]

## Razón de las decisiones

- Se ha usado Bootstrap y CSS propio para equilibrar velocidad de desarrollo con personalización visual, evitando una apariencia de plantilla genérica.[2]
- EmailJS permite que el formulario de contacto funcione en producción sin montar servidor, lo que simplifica el despliegue y hace el portfolio fácil de alojar en servicios estáticos como GitHub Pages o Netlify.[5][4]
- n8n se integra como ejemplo real de automatización y uso de agentes de IA, mostrando experiencia práctica en orquestar flujos entre diferentes servicios (chat, correo, APIs, etc.).[7][6]
